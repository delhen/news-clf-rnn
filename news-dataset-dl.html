Data Preprocessing¶

In data preprocessing, I did two tokenization processes. The first was tokenization which was made by myself. This was done to control some tokenizations that Keras Tokenizer cannot execute, i.e.:

  • Remove empty spaces that was found after punctuation tokenization. Example:
    ", Penunjang kualitas pendidikan" -> "  Penunjang kualitas pendidikan" -> "Penunjang kualitas pendidikan"
  • Remove watermark "Copyright 2018 , KLY KapanLagi Youniverse All Rights Reserved"
  • Trim more than one space and spaces at the beginning and end of paragraphs.

Deep Learning Model¶

For the model, I used the Embedding model with 200 data features. Then, I trained the embedding results with Recurrent Neural Network (RNN) combined with Long Short Term Memory (LSTM) of 64 layers. Then, I used Dense layer of 64 using ReLU as my activation function. After that, I used Dropout with a value of 0.2. I used Dense and Dropout for once again with each of them has 128 layers and a value of 0.5. Finally, I used Dense for 2 layers with softmax activation considering this is the output value issued. Then, I used Adam Optimizer to optimize my model and use categorical cross entropy as loss control. This can be done using binary cross entropy but considering I'm using a one hot encoder, I can't use this. For the complete model, you can see below.

model.summary()
